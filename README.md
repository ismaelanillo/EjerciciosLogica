## 🛠️ Requisitos e Instalación

Para ejecutar estos programas en tu máquina local necesitas un compilador de C++ (como `g++` mediante GCC o Clang).

```bash
# Clonar el repositorio
git clone [https://github.com/ismaelanillo12/EjerciciosLogica.git](https://github.com/ismaelanillo12/EjerciciosLogica.git)

# Entrar a la carpeta
cd EjerciciosLogica

# Ejemplo de compilación y ejecución de un archivo
g++ -o ejecutable Fibonacci.cpp
./ejecutable
